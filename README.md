# Currency Converter

A modern and responsive currency converter built with **HTML, CSS, and JavaScript** that provides real-time currency conversion using live exchange rates. The application features dynamic country flags, currency swapping, and a clean user interface designed for a smooth user experience.

## Live Demo

**Try the application here:**

[## Click Here](https://adityaingole1550.github.io/Currency-Converter---Made-by-me/)

---

## Features

* Real-time currency conversion using Exchange Rate API
* Dynamic country flag updates
* Currency swap functionality
* Responsive and modern UI
* Support for multiple world currencies
* Built with vanilla JavaScript (no frameworks)
* Built using Fetch API and ES6 Modules

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* Exchange Rate API
* Flags API
