# Currency Converter

A modern and responsive currency converter built with HTML, CSS, and JavaScript.

## Features

* Real-time currency conversion using Exchange Rate API
* Dynamic country flags
* Currency swap functionality
* Responsive and professional UI
* Support for multiple world currencies
* Built using Fetch API and ES6 Modules

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Exchange Rate API
* FlagsAPI

## Live Features

* Convert between currencies instantly
* Swap currencies with one click
* Automatically update country flags
* Display accurate exchange rates
